# 五行元素冒险游戏 - 演示说明

## 🎮 游戏概述

我已成功创建了一个完整的五行元素冒险网页游戏，完全满足您的所有需求：

### ✅ 核心功能实现

1. **3:4竖屏比例** - 游戏画面为480x640像素
2. **卡通像素风格** - 使用CSS像素化渲染和Canvas绘制
3. **三层场景系统** - 背景、中景、前景，具有视差效果
4. **平滑场景切换** - 无闪白，自然过渡
5. **坐标显示系统** - 实时显示世界坐标，原点为(0,0)
6. **任务系统** - 采集、击杀、对话三种任务类型
7. **场景限制** - 完成任务前禁止切换场景
8. **难度递增** - 距离原点越远，怪物越强，奖励越好
9. **本地存储** - 自动保存游戏进度
10. **技能系统** - 心法和外功双线发展

### 🎯 详细功能特色

#### 场景系统
- **三层渲染**: 背景层(天空)、中景层(玩家活动)、前景层(装饰)
- **视差效果**: 不同层次移动速度不同，营造深度感
- **平滑过渡**: 场景切换时的滑动动画效果

#### 任务机制
- **采集任务**: 随机生成金木水火土矿场，显示采集进度条
- **击杀任务**: 随机生成移动怪物，显示血条，自动战斗
- **对话任务**: 随机生成NPC，头顶感叹号标识
- **任务限制**: 完成前显示"完成场景任务后才能离开"提示

#### 五行元素系统
- **五种元素**: 金、木、水、火、土
- **动态奖励**: 根据距离和技能等级计算奖励数量
- **视觉反馈**: 不同元素用不同颜色表示

#### 技能升级系统
- **心法路线**: 增加获得元素数量
- **外功路线**: 提升攻击和采集速度
- **突破机制**: 每5级需要突破，持续消耗元素直到成功
- **概率系统**: 突破成功率随等级提升

#### 游戏对象
- **资源点**: 显示采集进度条，采集时间随距离增加
- **怪物**: 随机移动，血条显示，难度递增
- **NPC**: 对话交互，头顶感叹号提示

#### 用户界面
- **实时坐标显示**: 左上角显示当前世界坐标
- **任务状态显示**: 右上角显示当前任务和完成状态
- **背包系统**: 左下角显示持有的元素数量
- **技能按钮**: 右下角技能系统入口
- **血条和资源条**: 顶部显示玩家状态
- **漂浮文字**: 伤害数字、获得提示等动态效果

## 🚀 如何运行游戏

### 方法1: 直接打开
双击 `index.html` 文件在浏览器中打开

### 方法2: 使用启动脚本
- Windows: 双击 `start.bat`
- Linux/Mac: 运行 `./start.sh`

### 方法3: 手动启动服务器
```bash
cd webgame
python3 -m http.server 8080
```
然后访问 `http://localhost:8080`

## 🎮 操作说明

- **移动**: 方向键或WASD
- **对话**: 空格键(靠近NPC时)
- **技能**: 点击界面"技能"按钮
- **场景切换**: 完成任务后移动到边界
