# 五行元素冒险游戏

一个基于HTML5 Canvas的2D像素风格网页游戏，采用3:4竖屏比例设计。

## 游戏特色

### 🎮 核心玩法
- **三层场景系统**: 背景、中景、前景三层，具有深度感
- **场景切换**: 平滑的场景切换动画，无闪白效果
- **坐标系统**: 以原点(0,0)为中心的世界坐标系统
- **任务系统**: 每个场景随机生成采集、击杀、对话任务

### 🌟 游戏机制
- **五行元素**: 金、木、水、火、土五种资源系统
- **难度递增**: 距离原点越远，怪物越强，奖励越丰富
- **本地存储**: 自动保存游戏进度，刷新页面可继续游戏
- **技能系统**: 心法和外功两条升级路线

### 📊 技能详情

#### 心法 (Mind)
- **效果**: 增加每次获得元素的数量
- **升级**: 消耗五行元素进行升级
- **突破**: 每5级需要突破才能继续升级

#### 外功 (Body)
- **效果**: 加快攻击速度和采集速度
- **升级**: 消耗五行元素进行升级
- **突破**: 每5级需要突破才能继续升级

## 操作说明

### 移动控制
- **方向键**: ↑↓←→ 移动角色
- **WASD键**: 替代移动控制
- **空格键**: 与NPC对话

### 场景切换
- 完成当前场景任务后才能离开
- 移动到场景边界自动切换
- 切换时有平滑过渡动画

### 技能系统
- 点击"技能"按钮打开技能面板
- 选择心法或外功方向
- 升级需要消耗五行元素
- 突破需要持续消耗元素直到成功

## 游戏对象

### 🌳 资源点
- 随机生成金木水火土五种矿场
- 采集时间随距离增加
- 站在附近自动采集
- 显示采集进度条

### 👾 怪物
- 随机移动的敌人
- 血量和攻击力随距离增强
- 自动攻击范围内的怪物
- 击杀获得随机元素奖励

### 🤖 NPC
- 随机生成的非玩家角色
- 对话后获得元素奖励
- 头顶显示感叹号标识

## UI界面

### 状态显示
- **坐标显示**: 实时显示世界坐标
- **任务显示**: 当前任务和完成状态
- **背包显示**: 持有的元素数量
- **血条**: 玩家生命值（预留）
- **资源条**: 采集进度显示

### 特效系统
- **粒子效果**: 攻击、采集、死亡特效
- **漂浮文字**: 伤害数字、获得提示
- **过渡动画**: 场景切换平滑效果
